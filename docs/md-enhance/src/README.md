### Install

:::: code-group

::: code-group-item yarn

```bash
yarn add -D vuepress-plugin-md-enhance
```

:::

::: code-group-item npm

```bash
npm i -D vuepress-plugin-md-enhance
```

### Usage

:::: code-group

::: code-group-item ts

```ts
// .vuepress/config.ts
export default {
  plugins: [
    [
      "md-enhance",
      {
        // your options
      },
    ],
  ],
};
```

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "md-enhance",
      {
        // your options
      },
    ],
  ],
};
```
